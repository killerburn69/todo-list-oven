# Todo list Oven Technical Test

This is a basic todo app using NextJS

## File Structure
* app
    * components
        * TaskForm.tsx
        * TaskInput.tsx
        * TaskItem.tsx
        * TodoList.tsx
    * utils
        * utils.ts
    * interface.ts

## Features

- Add task
- Mark task when it is completed
- Deleta task


## Running Tests

To run todo app, run the following command

```bash
  yarn run dev
```
